HTML Elements Reference

An overview and Introduction

A HTML element is a type of HTML document component that defines the structure and content of a web page. Each element is represented by a tag, which can include attributes to provide additional information about the element. HTML elements can be nested within each other to create complex layouts and structures.

It is important to note that elements are not tags. Tags are the markup used to create elements, while elements are the entire structure that includes the opening tag, content, and closing tag. For example, in the element 


          <p> This is a paragraph.</p>
<p> is the opening tag, This is a paragraph. is the content, and </p> is the closing tag.

Tags are used to delimit the start and end of elements in the markup, The start and end tags of certain normal elements can be omited. The contents of the elements must be placed between just after the start tag(which may be implied in certain cases) and just before the end tag. For more info on this check W3C documentation

In the HTML syntax, most elements are written with a start tag and an end tag, with the content in between. An HTML tag is composed of the name of the element, surrounded by angle brackets. An end tag also has a slash after the opening angle bracket, to distinguish it from the start tag.


              <p>This is a paragraph.</p>

Types of HTML elements

There are three kinds of HTML elements:
Normal elements:
These usually have a start tage and an end tag although they can be omitted
Raw text elements
These are constructed with :
  • A start tag(in the form of <tag>)
  • Some amount of text content but no elements(all atgs , apart from the applicable end tag will be interpreted as content)
  • An end tag in which the elemnt name is prefixed with a slash i.e. </tag>
Void elements
Also called empty elements, single elements only have a start tags for example the <img> that is used to embed images inside the HTML document

  • Classification of HTML elements

    Block level elements vs inline elements

    Block level elements are elements that start on a new line and take up the full width available while inline elements are elements that do not start on a new line but only take up as much width as necessary
    Examples of block level elements include:
    • <div>
    • <p>
    • From <h1> to <h6>
    • <ul>, <ol>, <li>
    • <table>
    • <table>
    • <section>
    • <article>
    • <header>, <footer>, <header>
    Examples of inline elements include:
    • <span>Generic inline container
    • <a>Anchor(hyperlink)
    • <strong> for bold text
    • <img> For images
    • <em> For italic text

    Empty elements vs non-empty elements

    Empty elements are elements that do not have any content between the start and end tags. They are also called void elements. While non-empty elements have content between the start and end tags.
    Examples of empty elements include:
    • <br> for line breaks
    • <img> for images
    • <hr> for horizontal rules
    • <input> for form inputs
    • <meta> for metadata
    • <link> for linking external resources

    Media elements

    These elements are used to embed media such as images, audio, and video into a web page.
    These elements are:
    1. <img>
      used to embed images in a web page. It has the following attributes:
      • src: The URL of the image
      • alt: Alternative text for the image
      • width and height: Used to set the size of the image
      • loading: Used to control how the image is loaded (e.g., lazy loading)
    2. <audio>
      Used to embed audio files in a web page. It has the following attributes:
      • <src>Path to the audio file
      • controls: Adds playback controls (play, pause, volume, etc.)
      • autoplay: Automatically starts playing the audio when the page loads
      • loop: Repeats the audio when it ends
      • muted: Mutes the audio by default
    3. <video>
      Used to embed video files in a web page. It has the following attributes:
      • src: Path to the video file
      • controls: Display the pause, play and volume buttons
      • autoplay: Plays the video automatocally the page loads
      • loop: Repeats the video when it ends
      • muted: Mutes the video by default
